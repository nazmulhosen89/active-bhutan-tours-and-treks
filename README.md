# Active Bhutan Tours & Treks

Raw PHP website for a travel agency in Bhutan.

## Tech Stack
- PHP
- HTML
- CSS
- JavaScript

## Project Structure
- `index.php` / `index-english.php` → main pages
- `css/` → CSS stylesheets
- `js/` → JavaScript files
- `photogallery/` → images and gallery files
- `config.php` → configuration file (not included for security)

## Setup
1. Copy the project to `htdocs` (for XAMPP) or your PHP server
2. Create a database if required
3. Update `config.php` with DB credentials
4. Open in browser (e.g., http://localhost/activebhutantravelandtreks/)

## Note
- Sensitive files like `.env`, `config.php`, and uploads are ignored via `.gitignore`
